\# 📚 Library Student Fee Manager



एक सरल और प्रभावी उपकरण जो आपकी लाइब्रेरी के छात्रों की फीस को ट्रैक करने में मदद करता है।



\## ✨ Features



✅ \*\*Student Database\*\* - Photo के साथ पूरी जानकारी  

✅ \*\*Fee Tracking\*\* - मासिक फीस की स्थिति  

✅ \*\*WhatsApp Integration\*\* - स्वचालित याद दिलाने के संदेश  

✅ \*\*Dashboard\*\* - Statistics और Summary  

✅ \*\*Search\*\* - नाम या मोबाइल से खोजें  

✅ \*\*Responsive Design\*\* - सभी डिवाइसों के लिए  



\## 📋 Student Information Stored



\- Name (नाम)

\- Father's Name (पिता का नाम)

\- Mobile Number (मोबाइल नंबर)

\- WhatsApp Number (WhatsApp नंबर)

\- Email (ईमेल)

\- Address (पता)

\- Monthly Fee (मासिक फीस)

\- Photo (फोटो)

\- Joining Date (जॉइनिंग डेट)

\- Fee Payment History (फीस भुगतान इतिहास)



\## 🚀 How to Use



1\. Open `index.html` in a web browser

2\. Click "Add Student" to add new students

3\. Fill all required information and upload a photo

4\. View all students in "Students" tab

5\. Click on any student card to see full details

6\. Use "WhatsApp भेजें" to send reminders to students with pending fees

7\. Mark fees as paid using "फीस दर्ज करें" button



\## 📱 WhatsApp Integration



यह टूल WhatsApp Web API का उपयोग करता है। जब आप "WhatsApp भेजें" बटन दबाते हैं:



1\. एक नई विंडो खुलती है

2\. WhatsApp Web या App पर संदेश भेजने के लिए तैयार होता है

3\. आप "Send" बटन दबाकर संदेश भेज सकते हैं



\*\*Note:\*\* उत्पादन के लिए Twilio या WhatsApp Cloud API का उपयोग करें।



\## 💾 Data Storage



सभी डेटा आपके ब्राउज़र के LocalStorage में संग्रहीत है। इसका मतलब:

\- डेटा आपके कंप्यूटर पर सुरक्षित है

\- ब्राउजर का cache साफ न करें अन्यथा डेटा खो सकता है

\- नियमित बैकअप लें



\## 🔒 Privacy



सभी डेटा आपके ब्राउज़र में रहता है। कोई सर्वर पर नहीं जाता। पूरी गोपनीयता सुनिश्चित है।
